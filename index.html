Market Manager C++

Um sistema de gerenciamento de mercado robusto escrito em C++

Resumo

O programa tem como proposta principal ofertar uma solução aos problemas de manejamento de estoque de produtos que muitos mercados enfrentam no seu dia a dia. Utilizando de uma série de instruções claras e objetivas, o Market Manager é a ferramenta dos sonhos de todo gerente de estoque que eles nem imaginavam que precisavam.

Funcionalidades Principais

O sistema oferece diversas funcionalidades essenciais para o gerenciamento de um mercado:

  • Gerenciamento de Produtos (CRUD completo)
  • Gerenciamento de Clientes
  • Sistema de Vendas e Emissão de Recibos
  • Controle de Estoque
  • Interface de Texto (TUI) intuitiva
  • Persistência de dados (se aplicável)

Como Usar / Instalação

Para compilar e executar este projeto, você precisará de um compilador C++ (como g++ ou Clang). 

# 1. Clone o repositório e entre na pasta
git clone https://github.com/LucasFernandesCS/market-manager-cpp.git
cd market-manager-cpp

# 2. Compile o programa usando CMake
mkdir build
cd build
cmake ..
cmake --build .

# 3. Execute o programa (Exemplo para MacOS/Linux)
./main
